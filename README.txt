Compile with "javac *.java".
Use "java Import <file name>" to import a .csv file into the system.
Use "java Input <student name> <course name> <marks>" to input a student's course information.
Use "java CheckStudent <student name>" to check a student's information.
Use "java CheckCourse <course name>" to check a course's information.

Note:
1. After each operation, all information will be stored in a file named "Default.csv".
2. if the operation is not Import, information stored in "Default.csv" will be imported into the system automatically.
3. The test .csv file should be splited by just one English comma.
